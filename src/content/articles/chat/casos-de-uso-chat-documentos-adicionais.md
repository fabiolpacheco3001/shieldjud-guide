## Visão geral

O Chat do ShieldJud também permite utilizar **documentos adicionais de contexto** durante uma conversa.

Esses documentos podem ser enviados diretamente no chat para orientar análises ou comparações realizadas pelo sistema.

Entre os exemplos mais comuns estão:

- modelos de decisão  
- checklists de análise jurídica  
- decisões paradigmáticas  
- documentos de outros processos  
- orientações institucionais

---

## Elaborar minuta com base em um modelo

Um uso comum é enviar um modelo de decisão para orientar a elaboração de uma minuta.

Exemplo de solicitação:

"Elabore uma minuta considerando o modelo de decisão anexado."

Nesse caso, o sistema pode:

- analisar a estrutura do modelo enviado  
- utilizar os dados do processo como base  
- gerar uma minuta alinhada ao padrão indicado

Esse recurso ajuda a manter consistência no estilo decisório.

---

## Aplicar um checklist de análise

Também é possível enviar documentos contendo critérios de análise.

Exemplo:

"Realize a análise de admissibilidade considerando os critérios presentes no documento anexado."

O chat pode:

- interpretar os critérios do documento enviado  
- aplicá-los ao processo analisado  
- apresentar uma avaliação estruturada com base nesses parâmetros

---

## Comparar processos

O chat também pode ser utilizado para comparar o processo analisado com outro caso.

Exemplo:

"Compare este processo com o processo descrito no documento anexado."

A partir dessa comparação, o sistema pode identificar:

- semelhanças factuais  
- diferenças na narrativa jurídica  
- variações nos pedidos apresentados

---

## Inserir orientações institucionais

Também é possível utilizar documentos contendo diretrizes ou orientações institucionais.

Exemplo:

"Analise o processo considerando as orientações presentes no documento anexado."

Esse tipo de uso permite alinhar a análise do processo a critérios institucionais ou práticas específicas de uma unidade judicial.

---

## Benefícios desse recurso

O uso de documentos adicionais no chat permite:

- orientar análises segundo critérios específicos  
- comparar processos ou decisões  
- reutilizar modelos jurídicos  
- adaptar o comportamento do chat às práticas do usuário

Isso amplia a flexibilidade da análise assistida realizada pelo sistema.
